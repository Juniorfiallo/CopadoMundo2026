# Atualização v6.9.0 no GitHub

## Opção recomendada — versão completa

1. Descompacte `copa-2026-tracker-premium-v6.9.0.zip`.
2. Abra a pasta `copa-2026-tracker` criada pelo ZIP.
3. No repositório `CopadoMundo2026`, substitua os arquivos e pastas existentes pelo conteúdo dessa pasta.
4. Confirme que `index.html`, `app.js`, `styles.css` e `netlify.toml` permanecem na raiz do repositório.
5. Faça o commit com a mensagem: `Atualização premium v6.9.0`.
6. Aguarde o deploy automático do Netlify.
7. Abra o app e faça uma atualização forçada do navegador para renovar o cache.

## Opção rápida — patch

Descompacte o patch e envie seus arquivos mantendo a mesma estrutura de pastas. Os arquivos dentro de `data/`, `netlify/functions/` e `assets/` não devem ser movidos para a raiz.
