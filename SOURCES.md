# Fontes e integração

- Página oficial para conferência de classificação:
  `https://www.fifa.com/pt/tournaments/mens/worldcup/canadamexicousa2026/standings`
- Fonte automática padrão da versão 6:
  `https://worldcup26.ir/get/games`
  `https://worldcup26.ir/get/teams`
- Projeto open source da fonte padrão:
  `https://github.com/rezarahiminia/worldcup2026`

A fonte automática padrão é comunitária e não oficial. Para uso comercial ou missão crítica, substitua por um provedor esportivo contratado e configure as variáveis de ambiente descritas no README.
